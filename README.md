## 🔧 功能介绍

进入首页点击右下角红色机组按钮即可进入机组页面,

点击头部栏或者侧边栏即可返回机核主站.

由于是非官方组件,锚点每小时更新,

所以默认加载非最新机博,可以点击 [加载更新机博] 去加载更新的机博.
